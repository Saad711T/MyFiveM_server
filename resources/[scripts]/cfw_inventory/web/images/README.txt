ضع صور العناصر هنا. كل صورة باسم العنصر التقني + .png

المطلوب للمرحلة الحالية (العناصر التجريبية):
  pistol.png   - صورة المسدس
  water.png    - صورة الماء
  bread.png    - صورة الخبز
  money.png    - صورة المال/الفلوس

ملاحظات:
- الصيغة: PNG (يفضّل بخلفية شفافة)
- الحجم المثالي: 128x128 أو 256x256 بكسل
- لو الصورة ناقصة، الواجهة تعرض أول حرف من الاسم تلقائياً (ما يكسر شي)

لإضافة عنصر جديد لاحقاً:
1. أضفه في config.lua داخل Config.Items
2. حط صورته هنا باسمه: <name>.png
